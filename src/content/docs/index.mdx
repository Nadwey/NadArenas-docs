## Setup

### Requirements

- Java 21 or newer
- [PaperMC](https://papermc.io/software/paper) 1.20.6 or its forks
    - Older versions might work, but are not currently supported
- [WorldEdit](https://modrinth.com/plugin/worldedit/versions?l=bukkit)

### Installation

<Steps>

1. Make sure you meet the [requirements](#requirements)

2. Download the plugin from one of the following sources:
    - [Modrinth](#) (Recommended)
    - [SpigotMC](#)
    - [GitHub](https://github.com/Nadwey/NadArenas/releases)
    - TODO

3. Put the plugin in the `plugins/` folder of your server

4. And that's basically it, other plugins can now use NadArenas.

</Steps>
